# Ansible (IT PLAYBOOKS)
Une liste de playbook Ansible pour faciliter la tâche de ceux travaillant dans l'Informatique et Réseaux. 

# Proxmox-Cloning 
Ce playbook permet de cloner une machine virtuelle sur Proxmox en utilisant un template. Le playbook est interactif et demandera a l'utilisateur les paramètres nécessaires,
comme l'ID de la VM Template, l'ID de la VM clonée, quel node se trouve la VM, etc... 
Compatible avec la dernière version de Proxmox : 9.1.1 / RELEASE Nov 19 2025
